# Personal Income Tax Calculator (Python)

This project is a Python program that calculates the United States federal personal income tax
based on the 2009 tax brackets.

## Description
The program prompts the user to:
- Enter their filing status
- Enter their taxable income

It then calculates and displays the total tax based on the correct marginal tax rates.

## Filing Status Codes
- 0 – Single
- 1 – Married Filing Jointly or Qualified Widow(er)
- 2 – Married Filing Separately
- 3 – Head of Household

- 4. Follow the prompts on the screen
